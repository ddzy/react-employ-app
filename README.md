# react-employ-app

## Desc
- 具有实时聊天功能的招聘APP
- 前端(```React```, ```React-router-dom```, ```redux```, ```antd-mobild```, ```axios```)
- 后台(```MongoDB```, ```Socket.io```, ```Express```, ```Express```)
- 前后端连调采用 JWT 实现


## Dep
```
npm install
npm install -g nodemon
```

## Run
```
npm start

cd server & nodemon server.js
```

## Build
```
npm run build
```
